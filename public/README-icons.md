# Icônes PWA

Les fichiers suivants sont nécessaires pour la PWA :

- `pwa-192x192.png` - Icône 192x192px
- `pwa-512x512.png` - Icône 512x512px
- `favicon.ico` - Favicon
- `apple-touch-icon.png` - Icône Apple (180x180px recommandé)
- `mask-icon.svg` - Icône SVG pour Safari

**Note** : Ces fichiers sont des placeholders. Remplacez-les par vos vraies icônes avant le déploiement.

Pour générer les icônes, vous pouvez utiliser :
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
